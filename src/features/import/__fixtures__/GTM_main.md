## 助手

可以先按四个问题拆：

- 谁最痛
- 谁最容易现在买
- 谁最容易触达
- 谁最容易形成案例扩散

### 谁最痛

判断标准：问题是否高频、是否直接影响收入或交付、是否已有替代方案失效。

证据问题：他们最近一次因为这个问题损失了什么？现在怎么补救？谁在为此背锅？

常见误判：把“感兴趣”当成“真痛”，把老板口头支持当成真实优先级。

### 谁最容易现在买

判断标准：预算是否已经存在、采购链是否短、是否有明确项目窗口。

证据问题：本季度有没有预算口径？谁能拍板？当前是否有必须解决的 deadline？

常见误判：把长期合作意向当成当期购买意愿，把高客单价误判成高优先级。

### 谁最容易触达

判断标准：是否有现成名单、是否能低成本直达关键角色、是否能快速完成首轮访谈。

证据问题：我们手上有什么渠道？谁能介绍？触达后多久能拿到反馈？

常见误判：把曝光量当成有效触达，把关系浅的转介绍当成稳定渠道。

### 谁最容易形成案例扩散

判断标准：是否有可讲故事的结果、行业影响力是否足够、案例是否能被相似客户复制。

证据问题：这个客户的结果能否量化？是否愿意署名？能否影响同圈层客户？

常见误判：把名气大当成可扩散，把复杂定制项目当成标准案例。

# 结论

先不要急着选行业，先把判断标准、证据问题和常见误判框起来，再去补证据。
